Спецификация
1. Меню
  1.1 Кнопка "Кроссворды"
    1.1.1. При нажатии открывается страница с кроссвордами: вид - плитка, у каждого кроссворда: название, превью, сложность
    1.1.2. При нажатии на превью открывается страница с игровым полем
  1.2. Кнопка "Правила"
    1.2.1. При нажатии открывается страница с правилами: словесное описание правил игры с поясняющими картинками
2. Главная страница
  2.1. Главной страницей является страница с кроссвордами
3. Игровое поле
  3.1. Сетка, размер зависит от картинки и кратен блокам 5х5, разбита жирными линиями на блоки 5х5 клеток
    3.1.1. Загрузка матрицы картинки
    3.1.2. Создание сетки по размеру матрицы
  3.2. Сверху и слева от сетки цифры, показывающие кол-во подряд закрашенных клеток в столбцах и строках соответственно
    3.2.1. Анализ матрицы: подсчет подряд закрашенных клеток, выставление цифр в игровое поле
  3.3. Под сеткой кнопка "Проверить"
    3.3.1. Кнопка появляется после первой закрашенной клетки
    3.3.2. При нажатии происходит сравнение текущей сетки с матрицей картинки: при совпадении появляется надпись "Поздравляем!", при несовпадении появляется надпись "Попробуйте еще" и исчезает через 3 сек
4. Игровой процесс
  4.1. Закрашивание клетки по нажитию ЛКМ
  4.2. Стирание клетки по повторному нажатию ЛКМ
  4.3. Возможность зачеркивать цифры клеток, которые уже закрашены: при нажатии на цифру ЛКМ на ней появляется красный крестик
5. Подсказки
  5.1. Иконки подсказок находятся под сеткой
  5.2. Подсказка №1: показать неправильно закрашенную клетку
    5.2.1. При нажатии текущая сетка сравнивается с матрицей картинки, все закрашенные клетки, не совпадающие с оригиналом помещаются в массив, рандомно выбранная клетка показывается пользователю
  5.3. Подсказка №2: показать клетку, которая должна быть закрашена
    5.3.1. При нажатии текущая сетка сравнивается с матрицей картинки, все незакрашенные клетки, совпадающие с оригиналом помещаются в массив, рандомно выбранная клетка показывается пользователю
  5.4. На каждую картинку дается по 3 из каждой подсказки
  5.5. При нажатии на подсказку подсказанная клетка в течение короткого времени мигает зеленым цветом, чтобы привлечь внимание пользователя
