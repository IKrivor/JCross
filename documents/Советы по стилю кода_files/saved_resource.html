                                                                                                                                                  Насколько я понимаю это будет работать примерно так:
                                                                                                                                                  Из основного кода вызывается функция pow(x, n) и ей передаются некоторые параметры x и n. Эти x и n будут доступны внутри функции, как обычные переменные. Т.е. получается, что внутри функции мы проверяем являются ли переданный параметр n натуральным. Для этого используем внутреннюю функцию isNatural, которой передаем этот n и она возвращает true, если число натуральное и false если нет. Соответственно во тут:
                                                                                                                                                  if (!isNatural(n)) {
                                                                                                                                                  return NaN;
                                                                                                                                                  }
                                                                                                                                                  мы проверяем является ли n натуральным и если нет, то завершаем функцию возвращаем NaN. Сама эта проверка на n никак не влияет, оно не меняется.
                                                                                                                                                  Ну, а если оно натуральное, то идем дальше и строчкой "return doPow(x, n);" возвращаем из основной функции значение, которое получится при вызове вложенной функции doPow. И этой функции doPow мы передаем эти самые x и n, которые были переданы в основную функцию.
